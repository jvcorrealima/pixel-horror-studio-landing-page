# 🧟 Pixel Horror Studio - Portfolio Page

Bem-vindo ao portal da **Pixel Horror Studio**. Este projeto foi desenvolvido por mim (**Joao**) como parte dos meus estudos de **Front-end**.

## 🕹️ Sobre os Projetos

O site apresenta o conceito de dois jogos de terror originais:

1. **Bio-Rail: Last Stop**: Um survival horror claustrofóbico em um trem infestado por um vírus biológico, protagonizado pela recruta do FBI Helena Voss.
2. **Isla del Sangre**: Um pesadelo tropical em um resort abandonado e sangrento.

## 🛠️ Tecnologias Utilizadas

* **HTML5**: Estruturação semântica do conteúdo.
* **CSS3**: Estilização com foco em temática retrô/monospace e responsividade.
* **GIMP**: Edição e otimização das artes conceituais em Pixel Art.

## 🧠 Aprendizados

Neste projeto, pratiquei:
* Manipulação de cores e contraste em temas escuros.
* Alinhamento de elementos com Box Model.
* Otimização de imagens para web (conversão de .xcf para .jpg/.png).
* Narrativa aplicada ao design (Storytelling).
